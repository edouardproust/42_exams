# Exams training

## How to use

### Install grademe

- In file ~/.bashrc, add line `alias grademe='bash -c "$(curl https://grademe.fr)"'`
- Then in the current folder, run command `grademe`

OR

- In the current folder, run this command directly: `bash -c "$(curl https://grademe.fr)"`
